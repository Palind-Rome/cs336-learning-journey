## 这不是答案仓库

CS336 作业刻意只提供 tests 和 adapters，让你设计自己的实现。这里不会粘贴可提交的完整答案；会给出题目意图、接口契约、shape、伪代码、窄测试、失败症状和逐级提示。你仍需亲手完成实现并能解释每一行。

## 通用循环

1. 阅读 PDF 对应小题，写下输入、输出和不变量。
2. 打开 `tests/adapters.py`，确认测试真正调用的签名。
3. 预测一个最小样例的结果或 shape。
4. 写最朴素的正确实现，只跑一个窄测试。
5. 失败时先看第一个不同值、最大绝对误差、dtype/device 和广播轴。
6. 正确后再做性能、并行或大规模实验。

```bash
# 在对应 assignment 目录内
uv run pytest tests/test_model.py -k rmsnorm -v
```
